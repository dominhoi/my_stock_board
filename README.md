# 📊 My Stock Dashboard (나의 자산 및 매크로 시황 대시보드)

매일 아침 8시에 글로벌 매크로 지표, 관심 종목 및 보유 종목의 뉴스 분석, 국내 부동산 실거래 시황을 수집하여 포트폴리오를 브리핑하고 시각화하는 대시보드입니다.

## 🔗 실시간 대시보드 페이지
아래 링크를 통해 실시간 업데이트되는 포트폴리오 현황을 웹 인터페이스로 편리하게 확인하실 수 있습니다.

> ### 📈 [실시간 대시보드 바로가기](https://dominhoi.github.io/my_stock_board/)

---

## 🛠 주요 기능
1. **내 자산 현황 요약**: 총 매수 금액, 총 평가 금액, 종합 수익률 및 전일 대비 변동폭/변동률과 원/달러 환율 시각화.
2. **AI 에이전트 자산/부동산 제언**: 각 분야의 AI 컨설턴트들이 매크로와 포트폴리오 상태에 부합하는 투자 시나리오 제언.
3. **보유 및 관심 종목 뉴스 브리핑**: yfinance 최신 펀더멘털 뉴스를 Gemini AI가 단 1회 배치 처리하여 분석 및 링크 매핑.
4. **국내 부동산 거래 트래킹**: 서울, 분당, 용인 수지구 등의 최근 아파트 실거래가 및 매매/전세지수 트렌드 분석.

## 🧑‍💻 업데이트 방식
- `my_stock` 레포지토리의 GitHub Actions를 통해 매일 아침 8시(KST)에 파이썬 데이터 수집 스크립트가 실행됩니다.
- 수집 및 가공이 끝난 자산 데이터(`portfolio_data.json`)는 본 레포지토리(`my_stock_board`)의 `data.json` 파일로 자동 갱신 커밋 및 푸시되어 GitHub Pages를 통해 배포됩니다.
